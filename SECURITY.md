# Security Policy

## Supported Versions

Only the latest minor version is getting patches. If some major vulnerability occurs I might update older minor versions, but that is very unlikely to happen (I'm just one person)

## Reporting a Vulnerability

If you've found a vulnerability, please [create a new advisory it in the Security and quality tab](https://github.com/yesser-studios/yesser-todo-cli/security/advisories/new).
These will only be visible to you and the maintainer(s) by default until patched, when they will be published.
